Descrição do sistema:
 É um jogo da forca. O jogador tenta descobrir uma palavra secreta digitando letras. Quando acerta, a letra aparece na palavra. Quando erra, perde uma tentativa. O jogo acaba quando a palavra é descoberta ou quando as tentativas terminam.
Fiz esse tema porque é um jogo simples, divertido e permite usar vários conteúdos aprendidos em C++, como funções, repetições, condições e menu de opções.
 
Decisões técnicas:   
Usei funções para deixar o código mais organizado e fácil de entender. O menu fica repetindo até o usuário escolher sair. Usei um vetor para guardar a palavra do jogo e um for para verificar as letras digitadas. Também usei while para controlar a partida enquanto ainda houver tentativas.
Escolhi fazer um jogo da forca porque é um tema simples e permite usar vários conteúdos aprendidos durante as aulas de C++, como funções, laços de repetição, condicionais e menus.
